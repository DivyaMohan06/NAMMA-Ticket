# Namma Ticket

A public Next.js website for booking BMTC bus tickets, managing QR tickets,
checking Shakti eligibility, redeeming rewards, and purchasing bus passes.

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production

```bash
npm run build
npm start
```

The project uses the standard Next.js build output and can be deployed directly
to Vercel with the **Next.js** framework preset.
